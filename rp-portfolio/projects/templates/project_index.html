Space Invaders Game
This is a game created in Python using the Pygame library. Space Invaders is a fixed shooter in which the player controls a laser cannon by moving it horizontally across the bottom of the screen and firing at descending aliens. The player defeats an alien and earns points by shooting it with the laser cannon. As more aliens are defeated, the aliens' movement speeds up. If the cannon comes in contact with any of the aliens, the game finishes and the player loses. The game can be found here: Space Invaders

Classifying the asteroids as hazardous or non-hazardous
This is a machine learning project which I did as part of my internship with AITS. The dataset has been taken from Kaggle: NASA asteroids dataset. My job was to classify the asteroids as hazardous or non-hazardous by applying certain ML algorithms. I attained an accuracy of around 97% in the prediction. Post this, I also wrote a report on how I did the project, the problems I faced during the project, my conclusions, the results obtained, etc. The report was prepared strictly as per the IEEE format. The algorithm for the project can be found here: Algorithm for NASA asteroids classification
